## 画像取得ルール

1. **Unsplashのみ使用**（無料・商用利用可・クレジット推奨）
2. ファイル命名: `{prefecture}-{subject}-{連番}.jpg`
3. ダウンロード後に`data/image-credits.csv`へ即時記録
4. 重複チェック: PowerShellでファイルサイズ一致を確認
5. **Google画像検索から直接保存しない**
