## ファイル配置ルール

```
images/character/
├── base/         # 透過PNG・ポートレート系（背景なし）
│   └── noa-{description}.png
└── scene/        # 背景付きシーンイラスト
    └── noa-scene-{location/theme}-{variant}.png
```

- 透過PNG推奨（.png）
- base/: 最小 400×600px、推奨 600×900px
- scene/: 最小 1200×675px（16:9）
- ファイル名は英小文字・ハイフン区切り
